API con PostgreSQL

API para gestionar tickets de incidentes de dispositivos en un entorno laboral, conectada a PostgreSQL. Incluye métodos GET, POST, PUT y DELETE para consultar, crear, actualizar y eliminar incidentes.

Este proyecto fue desarrollado con un enfoque en la optimización y escalabilidad, asegurando que la API pueda manejar múltiples solicitudes simultáneamente. También incluye validaciones robustas para garantizar la integridad de los datos y una documentación clara para facilitar su uso.

Recomendaciones de Música

Para este proyecto se utilizó el API de Spotify para obtener recomendaciones de música basadas en un árbol binario. El Árbol Y base de Datos fue implementado usando Neo4j, una plataforma de base de datos gráficos.

El sistema permite a los usuarios descubrir música nueva basada en sus preferencias y patrones de escucha. Este proyecto combina algoritmos avanzados con una interfaz intuitiva, mostrando cómo la tecnología puede mejorar la experiencia musical.
